# Platform Character Limits

Current as of late 2026. Platforms shift these — verify in the active UI before bulk uploads.

## Google Ads — Responsive Search Ads (RSA)

| Field | Max chars | Count max |
|---|---|---|
| Headline | 30 | 15 |
| Description | 90 | 4 |
| Final URL | n/a | 1 |
| Display path | 15 (each of 2 segments) | 2 segments |

**Pin priority:** pin your strongest value prop to Headline 1. Leave the rest unpinned so Google can mix.

## Google Ads — Demand Gen / Discovery

| Field | Max chars | Count max |
|---|---|---|
| Headline | 40 | 5 |
| Long headline | 90 | 5 |
| Description | 90 | 5 |
| Business name | 25 | 1 |

## Google Ads — YouTube (Video action)

| Field | Max chars |
|---|---|
| Long headline | 100 |
| Description | 35 |
| CTA button text | 10 |

## Google Ads — Performance Max

| Asset type | Limit |
|---|---|
| Headlines | 5 short (30 chars) + 5 long (90 chars) |
| Descriptions | 1-5 (60-90 chars) |
| Images | 20 max, multiple aspect ratios |
| Videos | 5 (vertical, square, horizontal) |
| Logos | Square + landscape |

## Meta — Feed & Stories

| Field | Max chars | Visible before truncation |
|---|---|---|
| Primary text (caption) | 2200 | First 125 chars on mobile |
| Headline | 40 | Usually full |
| Description (link) | 30 | Usually full |
| CTA button | Preset options only | — |

**The first 3 lines of primary text** are what people read. Pack the hook there. Mobile truncates around 125 characters.

## Meta — Carousel

| Field | Limits |
|---|---|
| Primary text | 2200 chars |
| Per-card headline | 40 chars |
| Per-card description | 20 chars |
| Cards per ad | 2-10 |
| Image ratio | 1:1 (square) recommended |

## Meta — Reels / Stories Ads

| Field | Spec |
|---|---|
| Video duration | 1-60s (Reels), 1-15s per story card |
| Aspect ratio | 9:16 vertical |
| Safe zone | Keep critical text/faces inside 250px from top & 250px from bottom |
| Sound | Required — most viewers play with sound on (different from feed) |

## LinkedIn — Single Image / Single Video

| Field | Max chars |
|---|---|
| Introductory text | 600 |
| Headline (sponsored) | 70 |
| Description | 100 |

LinkedIn truncates intro text aggressively — first 150 characters matter most.

## Pinterest

| Field | Limits |
|---|---|
| Pin title | 100 chars |
| Pin description | 500 chars (first 50-60 visible) |
| Promoted pin image | 1000 × 1500 pixels (2:3) |

## Reddit Ads

| Field | Max chars |
|---|---|
| Headline | 100 |
| Body text (Post ads) | 40000 (basically unlimited) |
| Link / button text | n/a (uses Reddit defaults) |

## A practical rule

For every platform, write your headline to ~80% of the max char count. Tight headlines outperform character-stuffed ones, and you leave room for the platform's UI to render without truncation across devices.
